# PSIA-NRM Website — Editing Guide for Office Staff

Everything on the site that changes season to season is editable from the
WordPress admin (`/wp-admin`) — no developer needed. Log in and use this map.

## Where to edit what

| I want to change… | Go to… |
|---|---|
| Homepage hero text, buttons, or photo | **Site Content** (left sidebar) → Homepage Hero |
| Homepage stats (member count, school count, events) | **Site Content** → Homepage Stats |
| Homepage photo gallery | **Site Content** → Homepage Photo Gallery (pick from Media Library) |
| Footer blurb, office phone, office email | **Site Content** → Footer & Contact |
| The top navigation menu | **Appearance → Menus** |
| Dues table, scholarship summary, CEU rules, reinstatement fees, FAQ | **Pages → Membership** → edit the child page for that section (each section of the Membership page is its own page; order via Page Attributes → Order) |
| Any regular page (Scholarships, Contact, Awards, About, etc.) | **Pages** → edit it like a document. Tables and expandable Q&As are normal blocks. Search "NRM" in the block inserter for styled document links and deadline badges |
| Certification levels on a discipline page (prerequisites, documents, exam links) | Edit the discipline page → **Discipline Page Settings** box below the editor → add/edit levels and documents ("Media Library" button uploads PDFs) |
| A person in Who's Who (bio, role, school, photo) | **Members** → edit their entry (roles/disciplines/schools are the checkboxes on the right) |
| Events | **Events** → add/edit (date, location, price, registration link are fields below the editor) |
| PDFs and photos | **Media** → upload; then link to them from any page |

## Form submissions (contact, scholarships, grants)

- Every submission is saved under **Form Entries** in the admin — nothing is lost
  even if email notification hiccups.
- Export any form's entries to a spreadsheet: **Form Entries → Export CSV**.
- Notifications go to the office email set under Site Content → Footer & Contact.

## Rules of thumb

- **Publish/Update saves instantly to the live site.** Preview first if unsure.
- You can't break the design: layout, colors, and cards are locked in the theme.
  If a page ever looks odd after an edit, just undo (each page keeps revisions:
  the "Revisions" panel restores any earlier version).
- Don't delete pages named in the navigation without updating **Appearance → Menus**.
- New documents: upload to Media, then link — don't link to Google Drive for
  anything new (existing Drive links are historical).
